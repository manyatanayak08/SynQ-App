## 📱 About the Project
SynQ is a secure real-time Android messaging application built using Java. It allows users to send encrypted messages using Firebase backend services.

## 🚀 Features
- Real-time messaging
- User authentication (Firebase)
- Message encryption (Caesar Cipher)
- Profile image support

## 🛠 Tech Stack
- Java
- Firebase Authentication
- Firebase Realtime Database
- Android Studio

## ▶️ How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator/device
